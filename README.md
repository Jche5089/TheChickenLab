## The_Chicken_Lab
### 프로젝트 특징
- Machine Learning과 Web을 활용하여 소상공인들을 위한 치킨 프랜차이즈 브랜드 서비스 프로젝트
### 제공 서비스
- 소상공인들을 위한 서비스를 제공하는 차별화된 치킨 프랜차이즈 브랜드
   - 상권 분석을 통한 최적의 창업 입지 안내
   - 날씨에 따른 배달 주문량 예측을 통한 발주량 추천
